# Task 4 @ ElevateLabs

> ### Steps:
1. Initialize Terraform
**terraform init**

2. Plan the Infrastructure
**terraform plan**

3. Apply the Configuration
**terraform apply**

   - pulled and deployed the image in container : _subasangeeth/springapp:latest_

4. Verify that container is running
**docker ps**

5. Check Terraform State
**terraform state list**

6. Destroy 
**terraform destroy**

> ### Logs 

All Terraform command outputs (plan, apply, and destroy) are saved inside the logs/ folder.
